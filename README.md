The project focuses on responsive layouts, smooth animations, and reusable components created with Tailwind CSS.

# Features

ｏ Responsive and elegant UI using Tailwind CSS

ｏ Product slider using Splide.js

ｏ Auto-scrolling carousel animation

ｏ Typing animation using React Typed

ｏ Custom hover and corner border effects

ｏ Component-based structure for reusability

# Tech Stack

**React.js** – Frontend framework

**Vite** – Fast build tool for modern web apps

**Tailwind CSS** – Utility-first CSS framework for styling

**@splidejs/react-splide** – Lightweight and flexible carousel/slider

**@splidejs/splide-extension-auto-scroll** – Splide auto-scrolling extension

**React Typed (react-typed)** – Typing animation for text

**Lucide React (lucide-react)** – Icon library

**Framer Motion (framer-motion)** – Smooth animations and transitions

# Installed Libraries
# React Icons

npm i react-icons

# React Types Icons

npm i react-typed

# Tailwind Vite

npm install tailwindcss @tailwindcss/vite

# Latest version of the core Splide.js library

npm install @splidejs/react-splide

# for the Auto Scroll extension

npm install @splidejs/splide-extension-auto-scroll
